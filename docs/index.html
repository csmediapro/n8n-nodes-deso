n8n-nodes-deso

Set up a DeSo Identity credential for your n8n workflows.

1

Install the DeSo Identity extension

You need the DeSo Identity browser extension installed in your browser. It stores your wallet keys securely and handles signing.

2

Get your credential payload

Open the DeSo Auth Page in the browser that has the Identity extension. Click Connect DeSo Wallet, complete the authorization, then copy the credential payload.

Open DeSo Auth Page →
3

Paste into n8n

In your n8n DeSo Identity API credential, paste the entire JSON payload into the Credential Payload field. Save the credential — you're done.

Using the DeSo node

Name each credential

Use n8n's credential name to tell accounts apart, such as DeSo - Main or DeSo - Brand. If n8n creates a generic name, rename it from Settings → Credentials.

Text post

Add a DeSo node, choose the Post operation, select your credential, enter Post Text, and leave Image set to None.

Image from a previous node

Use this for images coming from a webhook, chat app, form upload, HTTP Request node, or agent workflow.

  1. Connect the image-producing node to the DeSo node.
  2. Set Image to From Previous Node.
  3. Set Image Field to the binary field name. Most n8n nodes use data.

Image from a URL

Set Image to From URL, paste a public image URL into Image URL, then run the node.

Privacy note: This auth page runs in your browser. It does not save, log, or transmit your DeSo credentials to CSMediaPro.
